# Primeiros Passos com Programação

Bem-vindo à sua jornada de programação. Este curso vai ensinar você a pensar como um programador e escrever código em múltiplas linguagens e paradigmas.

## O que é Programação?

Programação é o processo de escrever instruções que um computador pode seguir para resolver problemas. Cada aplicativo, site e ferramenta digital que você usa é construído sobre código escrito por programadores.

## Por que Aprender Programação?

- **Resolução de Problemas**: Divida problemas complexos em etapas pequenas e gerenciáveis
- **Oportunidades de Carreira**: Habilidades de desenvolvimento de software são muito procuradas em todos os setores
- **Criatividade**: Crie suas próprias ferramentas, jogos e aplicativos do zero
- **Automação**: Elimine tarefas repetitivas ensinando um computador a fazê-las para você

> [!NOTE]
> Aprender a programar é como aprender um novo idioma. Tenha paciência consigo mesmo e pratique todos os dias.

## Como os Computadores Entendem Código

Computadores entendem apenas binário — uns e zeros. Linguagens de programação como JavaScript, Python e Rust atuam como tradutoras. Você escreve código legível para humanos, e um compilador ou interpretador o converte em instruções de máquina.

## Seu Primeiro Programa

O primeiro programa tradicional exibe uma mensagem na tela:

```javascript
console.log("Hello, World!");
```

Esta única linha instrui o computador a exibir o texto `Hello, World!`. Cada linguagem tem sua própria maneira de fazer isso, mas o conceito é o mesmo.

## Exercício Prático

Modifique o programa Hello World para imprimir seu próprio nome:

```javascript
console.log("Hello, [Seu Nome]!");
```

Substitua `[Seu Nome]` pelo seu nome real e imagine a saída.
